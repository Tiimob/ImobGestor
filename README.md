# 🧩 ImobGestor

Ferramenta web desenvolvida com foco em automação e organização de rotinas operacionais.

## 🚀 Sobre o projeto

O ImobGestor foi criado com o objetivo de transformar atividades manuais em processos mais simples, padronizados e eficientes, utilizando uma interface leve e intuitiva.

A proposta é centralizar ações e facilitar o dia a dia, reduzindo erros e aumentando a produtividade.

## ⚙️ Funcionalidades

- Geração automatizada de informações
- Padronização de processos
- Interface simples e prática
- Organização de dados para uso interno

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript  

## 🌐 Acesse o projeto

👉 https://tiimob.github.io/ImobGestor/

## 📌 Objetivo

Este projeto foi desenvolvido como iniciativa prática para aprimorar conhecimentos em desenvolvimento web e criar soluções aplicáveis a cenários reais.

## 📈 Melhorias futuras

- Validação de dados
- Melhorias na experiência do usuário (UX)
- Integração com backend
- Armazenamento de informações
